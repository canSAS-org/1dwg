
Java (JAXB 2.1) binding for canSAS1d/1.1 XML standard 
for reduced small-angle scattering data.

$Id$


LICENSES

See LICENSES folder.


USE

To use the Java (JAXB 2.1) binding, all you need are 
the JAR files in the releases directory.
	http://svn.smallangles.net/svn/canSAS/1dwg/tags/v1.1/java

These are the files to download:
	cansas1d-1.1.jar
	cansas1d-1.1-javadoc.jar (optional)
	cansas1d-1.1-sources.jar (optional)


DOCUMENTATION

This is the Java binding for the cansas1d/1.1 XML 
standard for small-angle scattering data.

The code has been auto-generated using the JAXB v2.1
reference implementation from the cansas1d/1.1 XML schema.

The documentation for the code is available either:
* JAVADOC in JAR:  see above
* JAVADOC in HTML: 
**  The complete HTML is contained within the cansas1d-1.1-javadoc.jar above.  
    It can be extracted with any program that can extract a ZIP file.


DEVELOPMENT

To participate in the development of this code, checkout the ant-eclipse 
directory from this level. No further instructions are provided so you may
need to contact someone on the development team who understands this code.

svn co http://www.cansas.org/svn/1dwg/trunk/java/ant-eclipse ./cansas1d-java
